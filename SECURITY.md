# Security Policy

## Supported Versions

Sicherheitsfixes werden für die jeweils aktuelle Release-Version bereitgestellt.

## Meldung von Sicherheitsproblemen

Bitte keine Sicherheitslücken öffentlich als Issue posten.

Melde Sicherheitsprobleme bitte vertraulich an:

- GitHub: [Feberdin](https://github.com/Feberdin)

Bitte gib in der Meldung an:

- Betroffene Version
- Reproduktionsschritte
- Erwartetes vs. tatsächliches Verhalten
- Mögliche Auswirkung

Wir bestätigen den Eingang so schnell wie möglich und koordinieren den Fix.
